Eventum Javascript Style guide
==============================
Strict and elegant style guide for React Applications

Installation
------------
npm i eslint-config-eventum -D

Usage
-----
In your .eslintrc file, add:
```
"extends": ["eventum"]
```
or
```
"extends": ["eventum", "eventum/react"]
```
if you're also using React.
